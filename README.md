## Getting started

```bash
$ npm install --save @dalee/megafon-ui
```

or for `yarn`:

```bash
$ yarn add @dalee/megafon-ui
```

## Development notes

### Build

```bash
$ npm run build
```

### Interactive demo

```bash
$ npm run docz:dev
```

The demo will be available at [`http://localhost:3000/megafon-ui/`](`http://localhost:3000/megafon-ui/`).
