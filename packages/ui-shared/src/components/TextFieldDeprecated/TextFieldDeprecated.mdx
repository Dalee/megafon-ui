## API

| Method | Description                         |
| ------ | ----------------------------------- |
| blur   | Trigger blur event on input field.  |
| focus  | Trigger focus event on input field. |

**Use TextFieldDeprecated's 'ref' prop for API methods call:**

```
class SearchForm extends Component {
    this.textField = undefined;

    handleSubmit() {
        textField.blur();
    }

    render() {
        return (
            <Form onSubmit={this.handleSubmit}>
                <TextFieldDeprecated ref={(instance) => { this.textField = instance; }} />
            </Form>
        );
    }
}
```
