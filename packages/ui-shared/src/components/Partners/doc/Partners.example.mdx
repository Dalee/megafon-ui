
import { Playground } from 'docz';

## Примеры свойств
``` javascript

const items = [
    {
        href: '#',
        src: 'image.png',
    },
    {

    },
]
```

# Базовое использование

<Playground>
    <ContentArea>
        <Partners items={fourItems} />
    </ContentArea>
</Playground>

# Логотипы без ссылок

</Playground>

    </ContentArea>
