## Ошибка
Клик по иконки очищает поле по умолчанию. Если установлена иконка и `onCustomIconClick` то очищение перестает работать.

<Playground>
    <TextFieldWrapper>
        <TextField
            verification="error"
            noticeText="Name is required"
        />
    </TextFieldWrapper>
</Playground>

# Textarea
Пропс **multiline** выставленный в значение **true** изменяет компонент в `textarea`.

<Playground>
    <TextFieldWrapper>
        <TextField name="name" multiline placeholder="Multiline"/>
    </TextFieldWrapper>
</Playground>
