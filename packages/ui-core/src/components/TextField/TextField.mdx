## Mask usage

<Playground>
    <TextFieldWrapper>
        <TextField
            placeholder="+7 (999) 999-99-99"
            mask="+7 (999) 999-99-99"

</Playground>

## Error usage
1. On default settings clicking on the cross clears the field.
2. Set a custom icon and/or custom click handler disables the field clearing function

<Playground>
    <TextFieldWrapper>
        <TextField
            verification="error"
            noticeText="Name is required"
        />
    </TextFieldWrapper>
</Playground>

# Textarea view
The **multiline** property set to **true** changes TextField to textarea-like view.

<Playground>
    <TextFieldWrapper>

</Playground>
