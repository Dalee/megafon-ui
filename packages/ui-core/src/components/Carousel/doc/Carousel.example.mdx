## Особенности использования на мобильных разрешениях
На разрешении меньше 1024 отключаются стрелки и появляются "уши",
для корректного отображения которых необходимо предоставить компоненту возможность растянуться на всю страницу.
Чтобы основные слайды выравнивались по остальному контенту страницы,
необходимо переопределить размеры внутренних отступов у компонента с помощью
переданного в innerIndentsClass класса.
По умолчанию используются гайдовые отступы для контента страницы (32px на 320-767 и 48px на 768-1023)
