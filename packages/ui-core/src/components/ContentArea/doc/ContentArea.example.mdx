## Базовое использование

## Фоновый цвет внешнего контейнера
Фоновый цвет применяется ко всему компоненту

<Playground>
    <ContentArea outerBackgroundColor="spbSky2">
        Контент
    </ContentArea>
</Playground>

## Фоновый цвет внутреннего контейнера
Фоновый цвет применяется к отцентрированному контейнеру с ограниченной шириной.
Разница будет видна при ширине компонента больше 1440px

<Playground>
    <ContentArea innerBackgroundColor="spbSky2">
        Контент
    </ContentArea>
</Playground>
